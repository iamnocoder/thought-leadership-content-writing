# Changelog

All notable changes to the **write-sharp** skill are documented here.
This project uses [Semantic Versioning](https://semver.org): `MAJOR.MINOR.PATCH`.

- **MAJOR** — you changed the skill in a way that breaks how people already use it.
- **MINOR** — you added new guidance/capability, backward compatible.
- **PATCH** — small fixes, typos, clarifications.

## [1.0.0] — 2026-06-23

### Added
- Initial public release.
- Three-phase content standard: pre-write brief, drafting craft rules, post-write critique.
- Narrative framework: story structure, emotional chemistry, positioning, opening mechanics, voice/register.
- AI-pattern removal: passive voice, binary contrasts, throat-clearing, false agency, red flags.
- Optional brand/style-guide layer.
- Combined single-file build (`dist/write-sharp.md`) for non-Claude agent frameworks.
